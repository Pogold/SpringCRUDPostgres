# SpringCRUDPostgres
Students CRUD with database
Был создан api для работы с информацией о Студентах. 
# Функционал
- Создание Студентов
- Удаление Студентов
- Просмотр Студентов
- Изменение Студентов
- Хранение в БД
- Хранение в оперативной памяти
